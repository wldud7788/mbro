쇼핑몰에서 활용할 수 있는 QR코드 안내
QR코드 스캔 시 QR코드용 치환코드 (크기값는 1~10사이) 이용 안내 (오프라인에서는 인쇄물에 QR코드를 활용하세요) 바로가기
메인 페이지로 이동 {=qrcode("shop",3)} 1) 모든 페이지 : 치환코드 삽입 → QR코드 보임 설정 > 일반정보
상품상세 페이지로 이동 {=qrcode("goods",goods.goods_seq,3)} 1) 상품상세페이지 : 치환코드 삽입 → 해당 상품의 QR코드 보임
2) 그 외 페이지 : 상품고유번호를 넣은 치환코드 삽입 → 해당 상품의 QR코드 보임 		상품
카테고리 페이지로 이동 {=qrcode("category",categoryCode,3)} 1) 카테고리페이지 : 치환코드 삽입 → 해당 카테고리의 QR코드 보임
2) 그 외 페이지 : 카테고리고유번호를 넣은 치환코드 삽입 → 해당 카테고리의 QR코드 보임 		카테고리
브랜드 페이지로 이동 {=qrcode("brand",brandCode,3)} 1) 브랜드페이지 : 치환코드 삽입 → 해당 브랜드의 QR코드 보임
2) 그 외 페이지 : 브랜드고유번호를 넣은 치환코드 삽입 → 해당 브랜드의 QR코드 보임 		브랜드
이벤트 페이지로 이동 {=qrcode("event",event_seq,3)} 1) 이벤트페이지 : 치환코드 삽입 → 해당 이벤트의 QR코드 보임
2) 그 외 페이지 : 이벤트고유번호를 넣은 치환코드 삽입 → 해당 이벤트의 QR코드 보임 		이벤트
