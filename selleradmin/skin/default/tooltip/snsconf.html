페이스북 집접 연결

  • 좋아요 정보를 페이스북 API로 연결하여 전송합니다. 좋아요 버튼은 페이스북의 디자인으로 고정됩니다.

페이스북 간접 연결

  • 좋아요 정보를 별도로 전송 합니다. 또한, 좋아요 버튼의 디자인 변경이 가능하며, 좋아요 노출 속도가 향상됩니다.

페이스북 좋아요

  • 상품에 좋아요를 클릭하여 페이스북에 상품 정보를 노출 할 수 있습니다.
  • 좋아요를 클릭한 상품을 주문하는 구매자를 위한 추가 혜택을 설정할 수 있습니다.
  • 추가 혜택 적용 범위 자세히 보기

페이스북 쇼핑몰

  • 쇼핑몰에서 판매하는 상품을 페이스북에서 판매할 수 있습니다.
  • 페이스북 쇼핑몰 이용 시 유료보안서버 (SSL) 사용은 필수 입니다.
  • 연결된 페이스북 페이지는 facebook.com 에서 확인 가능합니다.
  • 페이스북 페이지 연결 해제는 developer.facebook.com 에서 가능합니다.
  • 페이스북 탭 추가 방법 자세히 보기

카카오 연동

  • 상품상세 페이지, 게시물보기 페이지, 이벤트 페이지에서 카카오톡으로 정보 공유 가능합니다. 단, 정보 공유 기능은 모바일 환경 전용입니다.
  • 카카오 앱 정보 공유를 사용하시려면 검색엔진최적화(SEO) 에서 페이지별 타이틀을 설정해주세요.

썸네일 레이아웃

  • 스킨에서 제공되는 인스타그램의 썸네일의 이미지 크기 및 썸네일 레이아웃을 설정할 수 있습니다.

네이버 톡톡 연동

  • 네이버 톡톡을 연동하시면 회원과 1:1 채팅이 가능합니다.
    • 네이버 톡톡 대화창 바로 연결
    • 상품 문의 요약 전달
    • PC 관리자와 Mobile 전용앱을 통한 고객 응대

스니펫

  • 상품 상세페이지에서 상품요약정보를 톡톡 대화를 시작할 때, 구매자에게 자동으로 말풍선으로 보낼 수 있는 기능

쇼핑몰 로고 이미지

  • 고객들이 네이버 아이디로 로그인 또는 회원가입 시에 고객정보 수집동의 확인 페이지에서 이미지가 제공됩니다.

쇼핑몰 로고 이미지

  • 네이버 아이디로 로그인 서비스를 재신청 하지 않고 관리자 권한을 공유할 수 있습니다.
  • 먼저 현재 네이버 아이디로 로그인 서비스 등록시 신청한 로그인 정보로 아래 페이지에 접속합니다.
    https://nid.naver.com/devcenter/main.nhn (네이버 로그인 필요)
  • 등록된 애플리케이션 아래 [멤버관리] 클릭
  • 양도하려는 관리자의 "네이버 아이디" 를 등록
  • 관리자 권한 양도(관리자 공유) 완료

카카오 맵

  • 주소/위치명 란을 비워두면 설정>상점정보>사업자정보>사업장주소/상호 가 표시 됩니다.
  • 지도레벨(줌)은 1에서 14까지 설정 가능 합니다. 지도레벨을 비워두면 기본 3으로 설정 됩니다.
