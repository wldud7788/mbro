해외배송상품 개인통관 고유부호

개인통관 고유부호는 개인 결제 시 구매자께서 직접 입력하시게 됩니다.
개인통관고유부호
· P로 시작하는 개인통관고유부호를 입력하세요.
해외배송상품은 관세청 통관 신고를 위해 구매고객의 고유식별정보인 개인통관고유부호를 수집합니다.
개인통관고유부호는 통관 시 주문등록번호 대신 사용 가능한 번호로, 관세청 사이트에서 발급 받으실 수 있습니다.
개인통관고유부호 발급안내 X
개인통관 고유부호는 해외배송 상품 통관 시 주민등록번호 대신
사용 가능한 번호로, 관세청 사이트에서 발급 받으실 수 있습니다.
개인통관고유부호 발급방법
1. 관세청 전자통관 사이트(https://p.customs.go.kr) 방문
2. 사이트 상단에서 업무처리 메뉴 선택
3. 좌측 메뉴에서 수입통관 메뉴 선택
4. 좌측 메뉴에서 개인통관고유부호 항목 선택
5. 본인 이름과 주민등록번호 입력 후 ‘확인’ 클릭
6. 공인인증서 인증
7. 주소, 전화번호, 휴대폰번호, 이메일 입력 후 최종 ‘등록’
8. 개인통관고유부호 확인
개인통관고유부호 수집 X
개인통관고유번호는 수입통관 업무처리를 위해 수집하며, 서비스 이용기간 동안 보관합니다.
개인통관고유부호 수집 X
