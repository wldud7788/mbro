1. 바코드 정보 입력 규칙

바코드 종류 규칙
숫자 영문 특수문자 길이제한 기타
Code39 가능 대문자 - SPACE $ / % 없음
Code128-A 가능 대문자 - SPACE $ / % 없음
Code128-B 가능 대문자 - SPACE $ / % 없음

방법 4. 정의된 코드로 바코드 입력 방법

상품 코드/정보에서 상품 속성(브랜드, 색상, 사이즈 등)에 대한 코드를 등록합니다.
미리 등록한 속성으로 상품 등록 시 바코드 정보가 자동 생성됩니다.
