출석 체크 방법

  • 스템프
    • 출석 체크 이벤트 페이지에서 [출석 체크] 버튼 선택 시, 출석 체크가 완료됩니다.
  • 댓글 작성
    • 출석 체크 이벤트 페이지에서 댓글 등록 시, 출석 체크가 완료됩니다.
  • 로그인
    • 쇼핑몰 로그인 시, 자동으로 출석 체크가 완료됩니다.

이벤트 페이지의 배너

  • 전체 이벤트 페이지의 배너 및 링크를 설정할 수 있습니다.
  • 위 이벤트 페이지는 전체 이벤트 설정 페이지에서 설정 할 수 있습니다.

혜택 설정

  • 고객에게 제공할 혜택을 설정해주세요.
  • 추가 혜택 범위 및 안내 자세히 보기

유입경로 URL

  • 입력한 URL 경로로 쇼핑몰 접속 시, 할인 혜택이 적용됩니다.
  • 단, 입력한 URL에서 경로가 변경된 URL을 경유하여 쇼핑몰에 접속하는 경우 유입경로 할인이 적용되지 않으니 주의하시기 바랍니다.

관리자 테스트

  • 테스트 방법: 관리자로 로그인 한 상태에서 [테스트] 버튼을 선택 합니다. 해당 버튼으로 쇼핑몰 접속 시 유입경로 할인이 임의로 적용됩니다.
  • 적용된 할인 내역은 상품 상세, 장바구니, 주문/결제 페이지에서 확인할 수 있습니다.

혜택 설정

  • 추가 혜택 범위 및 안내 자세히 보기

최소 주문 금액

  • 최소 주문 금액은 상품 쿠폰 적용 여부와 관계없이 전체 상품의 주문 금액 기준입니다.

중복 할인

  • 각 상품의 종류와 상품의 수량에 비례해 할인이 적용됩니다.

사은품 교환

  • 사은품 교환은 마일리지를 사은품으로 교환하는 이벤트 입니다.
  • 사은품 교환은 MY 페이지에서 사은품 교환(전용 스킨) 또는 혜택 교환(반응형 스킨) 신청할 수 있습니다.

페이지 진입 제한

  • 페이지 진입 ‘불가‘ 선택 시, 쇼핑몰 회원의 이벤트 페이지 진입이 제한됩니다.
  • 관리자로 로그인한 경우 설정 여부와 관계 없이 이벤트 페이지 진입이 가능합니다.

전체 이벤트 페이지

  • 전체 이벤트 페이지의 썸네일 및 링크를 설정할 수 있습니다.
  • 위 이벤트 페이지는 전체 이벤트 설정 전체 이벤트 설정 페이지에서 설정 할 수 있습니다.

상품 할인 혜택

  • 고객에게 제공할 혜택 금액을 설정해주세요.
  • 해당 이벤트가 단독 이벤트와 중복으로 진행되는 경우 단독 이벤트의 혜택이 우선 적용 됩니다.
  • 정가 기준으로 할인 이벤트를 적용하는 경우 반드시 상품 상세 페이지에서 정가를 입력해주세요.
