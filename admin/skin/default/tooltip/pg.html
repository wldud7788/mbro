페이코 계약진행 상태 안내

  • 신청전 : 페이코 온라인 신청을 하지 않은 상태
  • 신청접수 : PAYCO 온라인 신청을 한 상태
  • 계약대기 : 계약서와 구비서류가 파트너센터로 접수되지 않은 상태
  • 계약완료 : 서류접수가 접수(업로드)된 상태
  • 심사중 : PAYCO 서류심사 및 카드사 심사인 상태
  • 승인완료 : PAYCO 서류심사 완료 (PAYCO 설정 후 사용가능 상태 카드사 심사는 카드사별로 심사가 진행 되며 심사가 완료된 카드사는 사용가능합니다.)
※ 계약불가 : 심사 시 PAYCO 와 계약불가 사유 발생

계약진행 순서

  • 퍼스트몰에서 온라인 신청 (가맹점 신청)
  • PAYCO에서 신청정보를 토대로 신청 가맹점에 연락 및 안내
    - 계약조건 안내, PAYCO파트너센터를 통한 구비서류 업로드, 계약 동의 안내 등
  • 계약조건 이행
    - 보증보험 납부(대상 가맹점에 한함) / 구비서류 업로드, 계약 동의 처리
    - 구비서류는 PAYCO파트너센터에서 업로드 ( https://partner.payco.com )
    ※ 파트너센터 로그인 정보는 가맹점 신청 시 기재된 E-mail을 통해 안내
  • 카드사 심사 진행 (영업일 기준 D+7일 내외)
  • 승인완료
  • PAYCO코드 발급
    - PAYCO코드는 가맹점 신청 시 기재된 E-mail을 통해 안내
  • 퍼스트몰에서 PAYCO 설정 및 테스트 후 간편결제 사용
    - 단, 카드사 심사가 완료 시 결제가 가능합니다.
