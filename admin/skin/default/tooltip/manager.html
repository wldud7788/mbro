출고리스트 해당 기간 동안의 배송완료해야 하는 출고건수 = '출고준비' + '출고완료' + '배송중' 출고건수
반품리스트 해당 기간 동안의 반품처리해야 하는 반품건수 = '반품신청' + '반품처리중' 반품건수
환불리스트 해당 기간 동안의 환불처리해야 하는 주문건수 = '환불신청' + '환불처리중' 환불건수

판매자-매장간 정산

비밀번호 설정 규칙

  • 영문 대문자, 영문 소문자,숫자,특수문자 중 2가지 이상을 조합한 8~20자
  • 동일하거나 연속되는 문자,숫자 3자리 이상 사용 불가 영문 회원의 생년월일, 전화번호 사용 불가
  • 키보드 상 나열된 문자열 3자 이상 사용 불가
  • love, happy, password, test, admin 은 사용이 불가
  • 사용 가능한 특수문자: !#$%&()*+-/:=>?@[＼]^_{|}~

접속 허용 IP

  • 접속 허용 IP를 사용 시, 입력된 IP로만 관리자페이지에 접속할 수 있습니다.

접속 허용 휴대폰

  • 접속 허용 휴대폰을 사용 시, 해당 관리자는 입력된 휴대폰번호로 인증 시 관리자페이지에 접속할 수 있습니다.

로그인 페이지 접속허용 IP

  • 로그인 페이지 접속 허용 IP 사용 시, 입력된 IP에서만 관리자 페이지에 접속할 수 있습니다 (접속 허용 IP가 아닌 경우 에러페이지로 이동)

담당자

  • 판매자(입점사)는 자신의 실물상품과 티켓상품이 판매되면 그 사실을 SMS 및 이메일로 알 수 있습니다. 수신 여부는 통신판매중계자와 협의 해 주십시오

개인정보 항목 마스킹(*) 표시 처리

  • 주문서에 입력된 개인정보(이름,전화번호,휴대폰번호,이메일,상세주소) 항목이 마스킹(*) 처리 됩니다. 자세히 보기
  • 주문/출고 내역서 인쇄 시 개인정보 항목은 마스킹(*) 처리 됩니다.
  • 주문 상세 화면 배송 정보 변경이 제한 됩니다.
  • 반품 상세 화면 반품 정보 변경이 제한 됩니다.
  • 환불 상세 화면 문자 발송 휴대폰 번호 변경이 제한됩니다.
  • 주문/출고 엑셀 다운로드 시 개인정보 항목이 포함되어 있는 경우 다운로드가 제한됩니다.
