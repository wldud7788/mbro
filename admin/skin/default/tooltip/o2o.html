오프라인 매장 아이콘 안내

  • 반송지 : 배송비 상세에서 반송지로 등록된 매장
  • 매장안내 : 입력한 매장 안내를 쇼핑몰 스킨에 노출 중인 매장
  • 매장수령 : 배송비 상세에서 수령 매장으로 등록된 매장
  • POS 연동 : 오프라인 매장 POS 신청을 완료 후 POS를 사용 중인 매장
  • 대표매장 : 오프라인 매장 수정 페이지에서 대표 매장으로 설정한 매장

분류

  • 등록된 매장의 분류를 설정하여 매장을 구분할 수 있습니다.

매장 안내

  • 매장 안내 노출 설정 시, 쇼핑몰 스킨에서 매장 정보가 노출 됩니다.
  • 현재 홈페이지 스킨 사용 시 반영되며, 추후 모든 반응형 스킨에 반영될 예정입니다.
  • 홈페이지 스킨 설정은 디자인 > 스킨설정 에서 가능합니다.

대표 매장

  • 대표 매장으로 설정된 매장은 쇼핑몰 스킨의 메인에서 매장 정보가 제공됩니다. (홈페이지 스킨 사용 시)
  • 매장을 여러 개 설정한 경우 대표 매장이 가장 첫 매장으로 제공 됩니다.
  • 홈페이지 스킨 설정은 디자인 > 스킨설정 에서 가능합니다.

POS 연동

  • POS는 기존의 KICC POS를 신청한 고객을 위한 기능입니다.
  • 오프라인 매장의 POS와 쇼핑몰을 통합 관리 할 수 있습니다.
  • POS 신청 방법 자세히 보기 에서 가능합니다.

연동 정보

  • 점포코드와 포스키는 POS 계약 완료 시 KICC에서 발급합니다.
  • 점포코드와 포스키 입력 완료 시 연동키가 자동 발급됩니다.
  • 정포코드, 포스키, 연동키 발급이 모두 완료되어야 KICC POS와 연동이 가능합니다.
