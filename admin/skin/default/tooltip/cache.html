캐시 자동 생성

  • 캐시파일 자동생성은 캐시파일을 매시 30분경에 자동 생성하여 노출되는 상품 및 정보를 최신으로 자동 유지하게 합니다.
  • 캐시파일 자동생성 조건은 캐시파일이 생성이 가능(정적인 상품디스플레이)하며, 캐시파일을 사용 중일 때 자동 생성 가능합니다.
  • 자동생성 조건을 만족하지 못하면 자동생성 설정은 자동으로 해제됩니다.
  • 캐시파일 자동생성은 최대 20개까지 설정 가능합니다.

캐시파일 사용

  • 회원 또는 관리자 로그아웃 시 사용합니다.
