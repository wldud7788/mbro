상품 기본코드

  • 상품기본코드 정보를 상품별로 직접 입력할 수도 있지만 미리 등록된 생성규칙에 의해 자동으로 상품 기본코드를 생성할 수 있습니다.
  • 기본 코드 자동 생성 원리 안내

자주 쓰는 추가 정보

  • 자주 쓰는 상품의 추가정보(제조사, 수입자, 원산지 등)를 미리 등록하여 간편하게 이용할 수 있습니다.
  • 대표 카테고리: 카테고리별 코드 값은 카테고리에서 입력할 수 있습니다.
  • 대표 브랜드: 브랜드별 코드값은 브랜드에서 입력할 수 있습니다.
  • 상품 번호: 자동 생성 상품 코드의 중복을 방지할 수 있는 상품의 고유 값 입니다. (중복 되지 않음)

자주 쓰는 필수 옵션

  • 자주 쓰는 상품의 필수옵션(색상, 사이즈 등)를 미리 등록하여 간편하게 이용할 수 있습니다.

자주 쓰는 추가 구성 옵션

  • 자주 쓰는 상품의 추가구성옵션(함께 구매 가능한 상품)를 미리 등록하여 간편하게 이용할 수 있습니다.

상품 검색용 색상

  • 정의된 검색용 색상으로 상품을 검색할 수 있습니다. 설정 시 검색페이지에 자동 반영되며, 미 설정 시 보이지 않습니다.

수동기간

  • 쿠폰 상품의 유효기간(사용기간)을 수동 결정

자동기간

  • 쿠폰 상품의 유효기간(사용기간)을 자동 결정

날짜

  • 쿠폰 상품의 유효일(ex.공연일)을 결정

지역

  • 쿠폰 상품의 사용위치(주소)를 결정

색상

  • 상품의 색상을 색상표로 표현하고자 할 때

기본 코드 적용

  • 기본 코드 규칙 설정에서 설정한 내용에 따라 기본 코드를 일괄 업데이트 합니다.

재고가 있으면 판매

  • 상품 발송(출고완료) 후 재고가 0인 경우, 품절(또는 재고확보)상태로 변경되어 판매 중지됩니다.

가용 재고가 있으면 판매

  • 가용 재고란 현재 판매 가능한 상품 재고의 수를 말합니다.
  • 가용 재고는 설정한 가용 재고의 수보다 재고가 적은 경우 상품을 자동으로 품절(또는 재고확보)로 변경하여 판매를 중지합니다.
  • 가용재고 계산: 가용재고 = 상품의 재고 - 출고 예약량 - 불량재고

재고와 상관없이 판매

  • 상품 발송(출고 완료) 후, 재고가 차감 되어도 재고와 상관없이 판매합니다.
  • 결제취소/반품/주문처리 되돌리기 후, 재고와 가용재고가 증가되며 재고와 상관없이 판매합니다.

과세 여부

  • 판매 상품에 따라 과세 여부를 설정할 수 있습니다.
  • 국내 PG사와 계약 시 부가세율은 10%, 해외 PG세의 경우 부과세 0%로 적용됩니다. (해외 PG 거래는 수출로 간주되어 영세율 적용)

성인 인증

  • 성인 인증 사용 시, 19세 이상 회원만 상품을 조회할 수 있습니다.
  • 성인 인증은 휴대폰 또는 아이핀을 통해 본인인증이 가능합니다. 본인 인증은 본인 확인에서 가능합니다.

청약 철회

  • 청약철회 불가 선택 시 결제 확인 이후, 취소, 반품, 교환이 불가합니다.
  • 청약철회 방침은 주문에서 수정 가능합니다.

개인통관고유부호

  • 해외에서 구매 대행한 상품을 대한민국으로 배송 시 대한민국 관세청 통관신고를 위해
    주문 시 구매 대상자(구매대행요청자)의 개인통관고유부호를 수집합니다.

상품코드

  • 상품 코드는 생성 규칙은 상품 코드 에서 설정할 수 있습니다.
  • 등록한 상품 코드는 바코드 생성 시 사용 됩니다.

재고

  • 재고는 정상 재고와 불량 재고의 합입니다.

불량 재고

  • 불량 재고란 상품에 하자가 있어 고객의 반품/교환 신청으로 반품된 상품의 재고 입니다.

가용 재고

  • 가용 재고는 판매 가능 수량을 말합니다.
  • 가용재고 = 재고 – 출고 예약량 - 불량재고
  • 출고 예약량 = 결제확인 수량 + 상품준비 수량 + 출고준비 수량
  • 출고 예약량은 배송이 예정되어 있는 수량입니다.

안전 재고

  • 재고관리 버전(All-in-one)을 사용 중인 경우 재고 수량이 안전 재고 이하로 떨어지는 경우 자동 발주가 생성됩니다.

입점 마케팅 상품 상태

  • 상품 상태 전달 가능 항목
  • 네이버 쇼핑 / 다음 쇼핑하우: 신상품, 중고, 리퍼, 전시, 반품, 스크레치
  • 페이스북 마케팅: 신상품, 중고(전시,반품,스크레치), 리퍼
  • 상품 상태 안내
  • 리퍼: 제조사의 A/S를 거쳐 재 판매되는 상품
  • 전시: 매장 전시 및 피팅으로 사용감이 있는 상품
  • 스크레치: 상품 이동, 보관 중 발생한 손상으로 인해 상품 가치가 저하되거나 일부 하자가 있는 상품

별도 설치비 유무

  • 일반적으로 설치가 필요한 상품에 대해 , 설치비가 추가로 발생할 경우 ‘설치비 추가’ 선택
    (상품정보제공고시 >계절가전 (에어컨 /온풍기 )> 추가설치 비용 항목 등)
  • 설치비가 이미 가격에 포함되어 있거나, 추가 설치비가 필요하지 않은 상품일 경우 선택하지 않음
  • 기본 설치비가 추가로 부과되는 상품이나 ‘설치비 추가’ 를 선택 하지 않는 경우
    네이버에서 등록된 상품을 삭제 처리 될 수 있으니 주의 요망

병행수입/주문 제작

  • 주문제작 상품이란 판매 당시 제작이 진행되어 있지 않아서 제작에서 배송되기까지 일정 시일이 소요되는 상품을 의미합니다
  • 수공예 물품, 주문제작 가구 등이 이에 해당하며 인쇄물, 판촉물 등은 이에 해당하지 않습니다.

입점 마케팅 착불 배송비

  • 네이버 쇼핑, 다음 쇼핑하우에는 착불 배송 해당 여부만 전달됩니다.

재고에 따른 판매 상태

  • 재고와 상관 없이 판매하는 경우, 재고 여부와 상관 없이 판매 상태는 모두 정상 입니다.
  • 재고 1개 이상 있을 때 판매하는 경우, 재고가 없으면 자동 품절 처리 됩니다.
  • 가용재고가 n개 이상 있을 때 판매하는 경우, 설정한 값에 따라 가용재고보다 많은 경우 정상, 적은 경우 자동 품절 됩니다.
