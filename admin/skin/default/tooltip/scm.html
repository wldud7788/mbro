거래처 미지급 잔액

  • 거래처별 미지급 잔액이 있는 경우 입력합니다.
  • 미지급 잔액 입력 시 정산 금액을 맞출 수 있습니다.
  • 미지급잔액은 거래처 최초 등록 시 등록 가능하며, 등록 후 수정이 불가합니다.

사용 창고

  • 해당 매장에서 판매할 수 있는 재고는 선택한 창고의 재고입니다. 
  • 사용 창고 변경 시 해당 매장에서 판매되는 상품의 재고가 변경한 사용 창고 기준으로 재계산됩니다.

기본 출고 창고

  • 해당 매장에서 판매된 상품을 출고처리 할 때 선택한 창고가 기본으로 선택되어 있습니다.
  • 다른 창고로 변경도 가능합니다. 당연히 해당 창고의 재고가 감소합니다.

기본 반품 창고

  • 해당 매장에서 판매된 상품을 반품처리 할 때 선택한 창고가 기본으로 선택되어 있습니다.
  • 다른 창고로 변경도 가능합니다. 당연히 해당 창고의 재고가 증가합니다.

재고

  • 창고의 재고
    • - : 해당 창고에서 재고(옵션기준)가 '5'이상인 상품을 검색
    • - : 해당 창고에서 재고(옵션기준)가 '5'이하인 상품을 검색
    • - : 해당 창고에서 재고(옵션기준)가 '1'이상 ~ '5'이하인 상품을 검색
    • - : 해당 창고에서 불량재고(옵션기준)가 '5'이상인 상품을 검색
    • - : 해당 창고에서 불량재고(옵션기준)가 '5'이하인 상품을 검색
    • - : 해당 창고에서 불량재고(옵션기준)가 '1'이상 ~ '5'이하인 상품을 검색
  • 쇼핑몰 재고
    • : 해당 쇼핑몰에서 재고(옵션기준)가 안전재고보다 부족한 상품을 검색
    • : 해당 쇼핑몰에서 재고(옵션기준)가 안전재고보다 '5'이상 많은 상품을 검색
    • - : 해당 쇼핑몰에서 재고(옵션기준)가 '1'이상 ~ '5'이하인 상품을 검색
    • - : 해당 쇼핑몰에서 재고(옵션기준)가 '1'이상 ~ '5'이하인 상품을 검색
    • - : 해당 쇼핑몰에서 안전재고(옵션기준)가 '1'이상 ~ '5'이하인 상품을 검색

상품번호/바코드

  • 상품번호: 상품번호(red color)와 옵션번호의 조합
  • 바코드: 기본코드와 옵션코드의 조합해당 매장에서 판매된 상품을 반품처리 할 때 선택한 창고가 기본으로 선택되어 있습니다.

정보

  • 재고(불량) : 해당 쇼핑몰 사용창고(N개)의 재고수량, 재고 中 불량수량
  • 안전재고 : 해당 쇼핑몰의 적정재고수량
  • 정가 : 해당 쇼핑몰에서의 정가(부가세 포함)
  • 판매가 : 해당 쇼핑몰에서의 판매가(부가세 포함)

공급처 배송

  • 공급처 배송을 체크하시면 상품의 재고가 없는 상태로 상품출고가 가능하며 출고 시 출고수량 만큼 자동입고 후 출고됩니다.(단, 하단의 발주 정보 등록은 필수입니다.)

분류 코드

  • 재고관리용 상품 분류입니다.

구분

  • 기초조정 : 시스템 도입 시 기초재고
  • 입고조정 : 전산재고 실물재고와 상이할 때
  • 출고조정 : 전산재고 실물재고와 상이할 때
  • 불량폐기 : 불량재고 폐기 시

불량 폐기

  • 재고수량 감소 (자산 감소)
  • 표시용 ‘불량수량’이 있을 경우 → ‘0’으로 초기화
  • 불량 폐기 수량 입력 시 표시용 ‘불량수량’을 참고하세요

재고

  • 입고조정 창고의 현재재고

금액(KRW)

  • 원가 반영 금액

이동전

  • 출고 창고의 현재재고

이동전

  • 입고 창고의 현재재고

이동후

  • 출고 창고의 이동 후 재고

이동후

  • 입고 창고의 이동 후 재고

총평균법

  • 총평균법은 재고자산의 원가를 평가하는 방법 중 하나로 기초자산 금액에 일정기간의 매입합계금액을 합하고 이를 총수량(기초재고+매입수량)으로 나누어 평균원가를 산출하는 방식입니다.

미마감

  • 상하반기 수불은 해당 월이 마감된 후 다음달부터 확인 가능합니다.

미마감

  • 분기별 수불은 해당 월이 마감된 후 다음달부터 확인 가능합니다.

미마감

  • 월별 수불은 해당 월이 마감된 후 다음달부터 확인 가능합니다.

미마감

  • 평균원가는 해당기간 기말에 산출되며 이후 확인 가능합니다.

쇼핑몰 안전재고

  • 주문이 발생한 해당 쇼핑몰 해당 상품의 안전재고

거래처명

  • 거래처 선택에 따라 아래와 같이 거래 유형이 정의됩니다.
  • 원화 거래처 : 대한민국 거래
  • 외화 거래처 : 수입 거래 (FOB, CFR, CIF 등)

납기 일자

  • 원화: 납기일자
  • 외화: SHIPPING_DATE

결제 조건

  • 원화: 결제조건
  • 외화: PAYMENT_DATE

부가세

  • 매입 부가가치세

재계산

  • 부가세가 있는 상품의 상품가액에 부가세 포함가격을 입력하였을 때 발주가액을 재 계산합니다.

구분

  • 정규 : 자동 발주 상품을 기초로 한 발주서
  • 비정규 : 수동으로 상품을 발주한 비정규 발주서
  • 비정규[임] : 발주서 없이 입고된 예외처리 시 생성된 임의 발주서
