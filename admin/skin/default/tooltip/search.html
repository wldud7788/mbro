검색창 기본 검색어

  • 페이지별로 검색 창에 기본으로 제공되는 검색어를 설정할 수 있습니다.
  • 페이지에 여러 개의 기본 검색어 설정 시 해당 페이지의 기본 검색어는 랜덤으로 보여집니다.
  • 검색 창 기본 검색어 예시

인기 검색어

  • 검색 창에 제공 되는 인기 검색어 사용 여부 및 기간을 설정할 수 있습니다.
  • 인기 검색어 예시

인기 검색어 추천 상품

  • 판매 데이터가 있는 경우 구매수량, 구매 금액 순으로 20개의 상품을 랜덤으로 제공합니다.
  • 판매 데이터가 없는 경우 상품평 순, 장바구니 순, 위시리스트 순, 좋아요 순, 구매수량 순으로 20개의 상품이 랜덤으로 제공합니다.

검색어 자동 완성

  • 저장된 누적 검색 데이터로 검색어 자동 완성을 제공합니다.
  • 검색어 자동 완성 예시

검색어 추천 상품

  • 판매 데이터가 있는 경우 판매된 상품을 검색어(카테고리, 상품명) 분석, 구매수량 순, 구매금액 순으로 20개의 상품이 랜덤으로 제공합니다.
  • 판매 데이터가 없는 경우 상품을 상품평 순, 장바구니 순, 위시리스트 순, 좋아요 순, 구매수량 순으로 20개의 상품이 랜덤으로 제공 합니다.

구우편번호(6자리)로 (구)지번 검색

  • (구)지번 주소를 검색하며 도로명 주소는 저장 되지 않습니다.
